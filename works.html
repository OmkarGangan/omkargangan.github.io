Machine Learning

Mumbai House Rent Prediction

In this project, I have scrapped housing website using Selenium module of python and created my own data set. Read the csv file using Pandas and performed, Data cleaning, Exploratory Data Analysis(EDA), Visualization using Seaborn and Matplotlib and House Price Prediction using Sk-learn. Built various models like Linear Regression, Decision Tree Regressor, ensemble models like Random Forest Regressor, Ada boost, Gradient Boost, XG Boost and finally Voting Classifier. Developed web application using Streamlit module of python.

Tools & Technologies used: Python, Selenium, Pandas, Numpy, Seaborn, Matplotlib, Sk-learn, Streamlit.

Walk/Run Prediction

In this project, I have developed End to End Machine Learning Model using streamlit and deployed using Heroku. This project aims to detect whether the person is running or walking based on the sensor data collected from iOS device. I have used pandas for reading csv file and performing EDA, Visualization using Seaborn and Matplotlib, Model building and prediction using Sk-learn, Streamlit for web app and deployed using Heroku.

Tools & Technologies used: Python, Jupyter Notebook, Pandas, Seaborn, Matplotlib, Sk-learn, Streamlit, Heroku.
